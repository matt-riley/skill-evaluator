## Why this matters

`pnpm format:check` (`oxfmt --check .`) fails on all 8 source files, exiting with code 1. This means CI would always fail on the format gate, making the check useless. The fix is to run `pnpm format` once to apply the formatter's corrections, then `format:check` will pass on subsequent runs. If the formatter output is undesirable, the `.oxfmtrc.json` config should be adjusted instead.
